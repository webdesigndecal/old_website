Build your own website. A sexy one.
Web Design DeCal is a 2-unit DeCal for students who want to make a beautiful website.

Students will be introduced to various design philosophies in web design, including spacing, color, and positioning, which are essential to designing a beautiful and intuitive websites. We will also cover modern design trends like responsive design, flat UI, and skeumorphism. Using these philosophies, students will learn how to make a website using HTML 5 and CSS 3. We will cover Javascript and JQuery at the end of the course to make dynamic elements.

It is important to note that this course is a design only course - meaning we will not cover the server-side details like Django and PHP. By the end of the course, you will be able to design a website like those featured on Siteinspire. (http://www.siteinspire.com)

Design Section

In Design section, we will spend a lot of time looking at other well-designed websites, notably those featured on Siteinspire. We will start each class with a website of the day. Students are encouraged to suggest their own website of the day.

We will then spend about a quarter of the class time on the topic of the day. After an introduction to the topic, we will spend a half of the class on Case Study, looking at other websites that demonstrates the key principle covered in class.

Programming Section

In Programming section, students will learn how to use the theory that they have learned in a real application by learning web design techniques that involve the theory. For example, students learn spacing in week 2, and they will learn how to use spacing in CSS through CSS box models.

Students will then apply what they have learned in their final project, where students create a website of their choice, using the key principles learned in class. A key takeaway from this class is to develop one’s own design style, not just imitating others. Final project will play a instrumental role in this process.

Evaluation
This class is evaluated on a Pass/Non-Pass basis. 70 or above is a P, and below 70 is a NP. You will lose 5 pts for each unexcused absence.
